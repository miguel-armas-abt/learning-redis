> ### ▶️ Mostrar hashes 
> ```shell script
> keys *
> ```
----

> ### ▶️ Obtener valores asociados a un hash
> ```shell script
> HGETALL category:01
> ```
---

> ### ▶️ Borrar cache
> ```shell script
> flushall
> ```
----

> ### ▶️ Obtener valor asociado a una key
> ```shell script
> GET app_token
> ```
